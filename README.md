# Chigekwu Portfolio

Static portfolio starter built with Tailwind CDN, custom CSS, and small JavaScript animations.

## Files

- `index.html` - page content and structure
- `styles.css` - custom palette, curves, motion, cards
- `script.js` - scroll reveal and cursor glow

## Update before publishing

Replace placeholder links:

- GitHub
- LinkedIn
- Resume
- Goodreads
- Letterboxd

Upload these files to the root of your site, replacing your current `index.html`.
